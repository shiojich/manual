#Virtual box での共有フォルダの作成#

まず Windows でフォルダを作る. 名前は英数字にする. 
プロパティを開き, 共有のページの"詳細な設定"をクリックし, 一番上にチェックを入れる. 

次に Virtual box で共有フォルダの設定をする.
メニューにある"デバイス"をクリックし, 共有フォルダにある"共有フォルダ設定"をクリックする. 
"共有フォルダ"を選択した後, 右端の小さなアイコン(プラスのボタン)をクリック. 
"フォルダのパス"の欄の右にある下向き三角をクリックし, "その他"を選択.
はじめに Windows で作成したフォルダを指定.
"自動マウント"と"永続化する"にチェックを入れ, OKをクリック.
今, share という名前のフォルダを作ったとすると, そのパスは /media/sf_share である. Virtual box ではフォルダ名が sf_share となる. そうしたくなければ, 別の方法もある.

最後に, 共有フォルダにアクセスできるユーザを設定する.
$ sudo gpasswd --add {ユーザー名} vboxsf

再起動する. (しなくてもいいかも)

注意 1 : 
〜は sudoers ファイル内にありません. 云々と言われた場合は, 以下を実行.
$ sudo usermod -G sudo {ユーザー名}
確認のため,
$ sudo cat /etc/group | grep {ユーザー名}
として, 二行出力されれば良い.

注意 2 :
許可がありません. と言われてアクセスできないときは以下を確認.
Windows コントロールパネルの「ネットワークと共有センター」にある「共有の詳細設定」がすべて有効にチェックされているかどうか.
Windows の共有フォルダのプロパティを見る. 「セキュリティ」のページ下部にある詳細設定をクリックし, Users のアクセス許可エントリ がフルコントロールで許可されているかどうか.
